# Basic Pacman Game
my basic pacman game using pygame (older version of python works too!)
DOWNLOAD code.py for the actual version, and then, run the command: python code.py 
WHAT YOU NEED TO PIP INSTALL is pygame to install pygame, run the command, pip install pygame.
then, download code.py and run python code.py
NOTE that code.py will be in your DOWNLOADS section, so, you can either move the code.py to desktop, or, run the command, cd Downloads and also NOTE that the Downloads is case sensitive. you HAVE TO put the capital D for Downloads OR IT WILL RESULT TO AN ERROR!!!
If you do not have python installed, please do install it to get this project running
